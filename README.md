# 📋 Sign-up Form

![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=flat&logo=css3&logoColor=white)
![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=flat&logo=html5&logoColor=white)

A responsive sign-up page built from scratch with HTML and CSS. This is a project from The Odin Project's Intermediate HTML & CSS curriculum.

## ✨ Features

- **Split-layout design:** Full-height background image panel alongside a clean form section.
- **Real-time validation:** Password fields enforce strength rules via the `pattern` attribute.
- **CSS pseudo-class feedback:** Inputs respond to `:valid`, `:invalid`, and `:focus` states.

## 🔧 Key Learnings

- **CSS Cascade:** Conflicting margin and layout rules between nested elements caused unexpected spacing — debugging this sharpened my understanding of how the cascade resolves conflicts and why declaration order matters.
- **HTML Form fundamentals:** Semantic `<label>` + `<input>` relationships, native browser validation through `pattern` and `minlength`.

## 🚀 How to run locally

1. Clone this repository:
   ```bash
   git clone https://github.com/LucasAzevedoTheDev/sign-up-form.git
   ```
2. Open `index.html` in your browser.
